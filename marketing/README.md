# Vault Guard — static landing (M7)

`index.html` is a **self-contained** marketing page (no build step, no trackers).

## Publish options

1. **GitHub Pages**  
   - Easiest: enable Pages from branch `main` with folder **`/marketing`** (if your Pages settings allow a subfolder), or  
   - Move/copy `index.html` to **`docs/index.html`** and publish from `/docs`.

2. **Product site**  
   Copy `index.html` (and adjust the GitHub URL if you fork) into your CMS or static host at e.g. `vaultcompass.io/vault-guard`.

3. **Raw preview**  
   Open `index.html` locally in a browser to review copy and layout before deploy.

## After deploy

Add the public URL to the root **`README.md`** (“Live site”) and to repository **About → Website**.
